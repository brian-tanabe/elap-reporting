Generate's ELAP's source sheet template and monthly report

    Source sheet columns must appear in this order:
    1. Reporting Month: MM-YY
    2. Open Date: MM/DD/YY
    3. Client: Text
    4. Program: Text
    5. Referral Source: Text
    6. Legal Server #: Text
    7. Assigned: Text
    8. Legal Issue: Text
    9. Type of Service: Text
    10. Status: Text
    11. Date Closed: MM/DD/YY
    12. Notes: Text
    13. Court appearances: General/Number

    Please note, each case in a reporting month must represent the state of the case at the end of that month. This is the only way we're able to associate court appearances for client's whose cases span multiple months to the correct month. Dorthy does this well.
